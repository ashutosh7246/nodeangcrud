### Prerequists:
1. Node 10.16.3
2. Angular CLI 8.3.17
3. Nodemon (optional)

# Starting Application
## Installing dependencies
```
cd <dir>/nodeangcrud/
npm i
cd <dir>/nodeangcrud/angular/
npm i
```
## Server
```
cd <dir>/nodeangcrud/
node app.js or nodemon
```
## Client
```
cd <dir>/nodeangcrud/angular/
ng serve --port 5000 --open
```
